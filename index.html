Perceptron

Loosely based on biological neuron

x = input
w = parameter (vector)
h = activation function
y = output

Mulilayer Perceptron

MLP

Perceptrons grouped in layers

Backpropagation

Training the network

Cost function and derivative

\[c = \sum{(y_{i} - \hat{y_{i}})^{2}} \quad and \quad \frac{\partial{c}}{\partial{w_{i}}}\]

• Assign blame to each component and get weight deltas.

• Use a gradient based method (SGD, Adam) to minimize cost.

Playground

Deep Neural Networks

Promises

• Arbitrarily large learning capacity
• No need of hand tuned features

Problems

• More parameters ⇨ Hard to optimize
• Needs heavy computation power for large enough network

Solutions

• Better architectures and techniques.
• Moore's law. General Purpose GPUs.

Layers

  • Dense

    Connect every neuron from last layer to this layer.

  • Convolution

    Use convolution to connect only locally dependent regions.

  • Recurrent

    Condition current output on past state.

  • Others

    Variations, attention, implementation tricks etc.

Convolution layer

• Use a bunch of kernels and sweep over input.
• Kernel weights are learnable.

Convolution layer

• Learn higher level features as they go deeper
• Lot less parameters to solve for than dense layer
• Works great in cases with inherent spatial invariance

Recurrent layer(s)

• Neuron output depends on current input and past state
• Used in language modelling and other sequence related problems

Libraries

Allow users to represent computation flow in terms of tensors and layers and provide automatic differentiation and execution on CPU / GPU

Keras example

from keras.models import Sequential
from keras.layers import Dense, Activation

model = Sequential()
model.add(Dense(output_dim=64, input_dim=100))
model.add(Activation("tanh"))
model.add(Dense(output_dim=10))
model.add(Activation("softmax"))

model.compile(loss="categorical_crossentropy", optimizer="sgd")
model.fit(X_train, Y_train, nb_epoch=5, batch_size=32)

classes = model.predict_classes(X_test)
proba = model.predict_proba(X_test)
